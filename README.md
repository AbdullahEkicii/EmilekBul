# 📱 EmilekBul

**EmilekBul**, kullanıcılara yapay zeka destekli test ve soru-cevap deneyimi sunan modern bir Flutter uygulamasıdır.  
Eğlenceli arayüzü, dinamik kategori sistemi ve kişiselleştirilmiş içeriklerle hem öğretici hem etkileşimlidir.

---

## 🚀 Özellikler

- ✅ **Kategori bazlı testler:** Zorluk seviyesine göre ayrılmış testler
- 🧠 **Yapay zeka destekli içerik üretimi:** Sorular OpenAI API ile dinamik olarak üretilir
- 📊 **Sonuç analizi:** Test sonrası kullanıcı performansı detaylı şekilde analiz edilir
- 💾 **Offline destek:** Geçmiş test sonuçları cihazda tutulur
- 📱 **Modern ve kullanıcı dostu UI**
